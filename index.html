Dodaj, edytuj i sprzedawaj przedmioty. To tylko front-end demo — podłącz własne API lub plugin do serwera.

Jak to działa?

  1. Gracz wybiera item i dodaje go do koszyka.
  2. Na checkout pokazujemy instrukcję (np. komendę /give lub kod w panelu serwera).
  3. W praktyce: po otrzymaniu płatności serwer przyznaje przedmiot lub administracja potwierdza transakcję.
